# 10.32 LAB: Warm up: People's weights (Vectors)

## Input
Create a file called `input.txt` with the following as the content:
```
236.0
89.5
142.0
166.3
93.0
```
Read in all the weight into a `vector` called `weights`.

## Output
```
Total weight: 726.80
Average weight: 145.36
Max weight: 236.00
```
